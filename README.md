# LiveBridge OBS Plugin

Connect OBS to Facebook and go live without managing stream keys, tokens, or API setup.

## Overview

LiveBridge is a cloud-backed service with an OBS plugin that allows users to connect their Facebook Page, automatically create livestreams, and go live reliably without managing stream keys, tokens, or API workflows.

Built for reliability, LiveBridge removes the most common causes of failed Facebook livestreams, including expired tokens, incorrect stream keys, and manual setup errors.

## Planned Features

- OBS stream configuration
- One-click Go Live
